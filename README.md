# FIFA World Cup 2026 PDF-Style Live Bracket

This version updates the design to visually match the attached bracket PDF:

- Centered trophy/header
- Green/red/gold stage tabs
- Left/right symmetrical knockout chart
- Navy match headers
- Gold final box and brown third-place box
- Footer and print styling

Default mode shows real qualified teams only. Click **Fill prediction** to show the prediction bracket. Click **Real teams only** to reset.

## Live API
Deploy to Netlify and add:

```txt
API_FOOTBALL_KEY=your_key
API_FOOTBALL_LEAGUE=1
API_FOOTBALL_SEASON=2026
```
